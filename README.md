# Online Coffee Shop

This repository contains a simple test project for an online coffee shop website built using basic web technologies.  
It is intended for learning, experimentation, and practicing JavaScript logic.

## Features
- Coffee product listings with images and prices
- Add and remove items from the shopping cart
- Real-time updates of total items and total price
- Clear cart and submit purchase actions

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

## Purpose
This project is for testing and educational purposes only.  
It is not intended for production use.

You are free to use, modify, copy, or reuse any part of this repository.
